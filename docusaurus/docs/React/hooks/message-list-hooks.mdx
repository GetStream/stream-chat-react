title: MessageList Hooks
---

The `MessageList` and `VirtualizedMessageList` components use a variety of custom hooks to assemble a scrollable list of `Message` components, with all of the necessary data and handlers provided to each.
These hooks can be useful when building a custom list component.

### useEnrichedMessages

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useEnrichedMessages.ts) that determines which messages need date separators and group css styles, returns the processed array of messages.

### useGiphyPreview

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useGiphyPreview.ts) that handles state setting logic for the optional `GiphyPreviewMessage` component in the `VirtualizedMessageList`.

### useLastReadData

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useLastReadData.ts) that returns a memoized value of read states for each message. Used within the `useEnrichedMessages` hook.

### useMessageListElements

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useMessageListElements.tsx) processes each message and returns the html elements needed to render each one, returned as an array.
This hook also passes all of the `Message` component props appropriately.

### useMessageListScrollManager

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useMessageListScrollManager.ts) that handles the list scroll logic, returns a handler function that calculates a new scroll top value.

### useNewMessageNotification

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useNewMessageNotification.ts) used in the `VirtualizedList` component and returns message notification information.

### usePrependMessagesCount

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/usePrependMessagesCount.ts) used in the `VirtualizedList` component and returns the number of messages prepended.

### useScrollLocationLogic

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useScrollLocationLogic.tsx) used to handle the scroll location logic of the list and returns the related data and handlers.

### useShouldForceScrollToBottom

A [custom hook](https://github.com/GetStream/stream-chat-react/blob/master/src/components/MessageList/hooks/useShouldForceScrollToBottom.ts) used in the `VirtualizedList`, returns a boolean for whether the list should be forced to scroll to the bottom.
